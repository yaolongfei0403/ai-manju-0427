## 1. 页面结构

### 1.1 组件树

```
LoginPage
├── StarfieldBackground          [展示型]  星空粒子动画背景
├── BrandingPanel                [展示型]  左侧品牌介绍（标语 + 特性列表 + 数据统计）
├── AuthCard                     [交互型]
│   ├── TabSwitcher (登录 | 注册) [交互型]
│   ├── LoginForm                [交互型]  邮箱/手机 + 密码 + 记住我 + 忘记密码
│   ├── RegisterForm             [交互型]  邮箱 + 验证码 + 密码 + 确认密码 + 协议勾选
│   └── OAuthButtons             [交互型]  微信 / QQ / GitHub 第三方登录
└── Footer                       [展示型]
```

### 1.2 全局布局位置

- **Layout**: `AuthLayout` — 登录注册专用布局
- **Header**: 无（登录注册页使用独立布局，不使用 GlobalHeader）
- **路由**: `/login`, `/register`, `/forgot-password`

### 1.3 视觉特征

- **背景**: 全屏深色渐变 (`#0f172a → #1e1b4b → #312e81`) + Canvas 星空粒子动画（`starfield`）
- **布局**: 左半屏品牌面板（渐入动画 `slideInLeft`）+ 右半屏登录卡片（毛玻璃）
- **品牌面板**: 大标语 "制作一部漫剧只需要一杯咖啡的时间" + 4 个特性图标（`fas fa-bolt` / `fas fa-clapperboard` / `fas fa-film` / `fas fa-wand-magic-sparkles`）+ 3 个数据统计（10万+ / 50万+ / 98%）
- **分割线**: 渐变透明线
- **OAuth 按钮**: 微信绿 / QQ 蓝 / GitHub 暗，`rounded-xl` + 图标 + 文字

## 2. 交互行为

### 2.1 操作列表

| # | 操作 | 触发条件 | 影响模块 | 预期结果 | 行为分类 |
|---|------|----------|----------|----------|----------|
| 1 | 登录 | 填写邮箱+密码，点击登录 | AuthCard | 验证身份，跳转项目列表 | CRUD |
| 2 | 注册 | 填写邮箱+验证码+密码，点击注册 | AuthCard | 创建账号，自动登录 | CRUD |
| 3 | 第三方登录 | 点击微信/QQ/GitHub | OAuthButtons | OAuth 授权跳转 | 导航 |
| 4 | 获取验证码 | 点击"获取验证码" | RegisterForm | 发送短信/邮件验证码 | CRUD |
| 5 | 记住我 | 点击复选框 | LoginForm | 延长 Token 有效期 | CRUD |
| 6 | 忘记密码 | 点击"忘记密码？" | AuthCard | 跳转密码重置流程 | 导航 |
| 7 | 重置密码 | 输入邮箱+验证码+新密码 | AuthCard | 重置密码成功，跳转登录 | CRUD |
| 8 | 切换登录/注册 | 点击 Tab | TabSwitcher | 切换表单视图 | 导航 |

### 2.2 表单校验规则

**登录表单**:
- 邮箱: 必填，有效邮箱格式
- 密码: 必填，最少 8 字符

**注册表单**:
- 邮箱: 必填，有效邮箱格式，唯一性校验
- 验证码: 必填，6 位数字，5 分钟有效
- 密码: 必填，8-32 字符，包含大小写字母和数字
- 确认密码: 必填，与密码一致
- 协议勾选: 必选

## 4. API接口

### 4.1 认证模块接口

#### POST /api/auth/register

用户注册。

```json
// Request
{
  "email": "user@example.com",
  "verification_code": "123456",
  "password": "Abc12345",
  "confirm_password": "Abc12345",
  "agreed_to_terms": true
}

// Response 201
{
  "user": { "id": "uuid", "email": "user@example.com", "username": "用户xxx" },
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIs..."
}
```

#### POST /api/auth/login

用户登录。

```json
// Request
{
  "email": "user@example.com",
  "password": "Abc12345",
  "remember_me": true
}

// Response 200
{
  "user": { "id": "uuid", "email": "user@example.com", "role": "admin" },
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIs..."
}
```

#### POST /api/auth/send-verification-code

发送验证码。

```json
// Request
{ "email": "user@example.com", "type": "register | reset_password" }

// Response 200
{ "message": "验证码已发送", "expires_in": 300 }
```

#### POST /api/auth/oauth/{provider}

第三方 OAuth 登录（wechat/qq/github）。

```json
// Request
{ "code": "oauth_authorization_code", "redirect_uri": "https://app.example.com/callback" }

// Response 200
{
  "user": { "id": "uuid", "email": "user@example.com" },
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "is_new_user": false
}
```

#### POST /api/auth/forgot-password

发起密码重置（发送验证码）。

```json
// Request
{ "email": "user@example.com" }

// Response 200
{ "message": "重置验证码已发送到您的邮箱", "expires_in": 300 }
```

#### POST /api/auth/reset-password

确认重置密码。

```json
// Request
{
  "email": "user@example.com",
  "verification_code": "123456",
  "new_password": "NewAbc12345",
  "confirm_password": "NewAbc12345"
}

// Response 200
{ "message": "密码重置成功，请使用新密码登录" }
```

#### POST /api/auth/refresh-token

刷新 JWT Token。

```json
// Request
{ "refresh_token": "eyJhbGciOiJIUzI1NiIs..." }

// Response 200
{
  "token": "eyJhbGciOiJIUzI1NiIs... (new)",
  "refresh_token": "eyJhbGciOiJIUzI1NiIs... (new)"
}
```

## 5. 业务能力

### 5.1 核心能力: S1 用户认证与授权

| 属性 | 说明 |
|------|------|
| **功能** | 邮箱/手机 + 密码登录、邮箱/短信验证码注册、第三方 OAuth（微信/QQ/GitHub）、JWT Token 管理 |
| **涉及 AI** | 无 |
| **异步处理** | 无 |

### 5.2 错误码（认证相关）

| 错误码 | 说明 |
|--------|------|
| `AUTH_INVALID_CREDENTIALS`(1001) | 用户名或密码错误 |
| `AUTH_TOKEN_EXPIRED`(1002) | Token 已过期 |
| `AUTH_EMAIL_ALREADY_EXISTS`(1003) | 邮箱已被注册 |
| `AUTH_VERIFICATION_CODE_EXPIRED`(1004) | 验证码已过期 |
| `AUTH_OAUTH_FAILED`(1005) | OAuth 授权失败 |

## 6. 关键用户路径

### 路径 A：新用户首次注册创作

```
注册/登录 → 项目列表 → 创建新项目（配置名称/风格/模型）
→ 上传小说文件 → 确认免责 → 配置分集策略 → AI 智能拆解
→ 查看/调整分集结果 → 编辑资产提示词 → 批量生产资产参考图
→ 确认入库 → 分集工作台（逐集编辑分镜描述 + 关联资产 + 生成分镜图）
→ 提交进入视频工作台 → 逐镜生成视频 → 时间轴编排 → 导出成片
```

### 路径 E：密码重置

```
登录页 → 点击"忘记密码" → 输入邮箱 → 获取验证码
→ 输入验证码 + 新密码 → 重置成功 → 跳转登录页
```
